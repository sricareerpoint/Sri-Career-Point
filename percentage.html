📊 Percentage

SSC CHSL Quantitative Aptitude

What is Percentage?

Percentage means a value out of 100.

Percentage = (Value / Total Value) × 100

Important Formulas

1. Percentage = (Part / Whole) × 100

2. Value = (Percentage × Total) / 100

3. Percentage Increase = (Increase / Original Value) × 100

4. Percentage Decrease = (Decrease / Original Value) × 100

Important Fraction to Percentage

1/2 = 50%

1/3 = 33.33%

1/4 = 25%

1/5 = 20%

1/8 = 12.5%

1/10 = 10%

Shortcut Example

Question: Find 25% of 200.

25% = 1/4

200 ÷ 4 = 50

Answer = 50

Remember common fraction-percentage values to solve questions faster.
